# Complaint Management System

A modern, Glassmorphism-based Complaint Management System built with Laravel and MySQL.

## Features
- **Public Complaint Submission:** Translucent, user-friendly forms for easy filing.
- **Admin Analytics Dashboard:** Sleek grid layout featuring real-time circular progress tracking for active and resolved cases.
- **System Activity History Log:** Secure, color-coded auditing of platform creations, updates, and deletions.
